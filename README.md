# 诱人的JWT
## 一、新建springboot项目
    主要依赖：
    h2、security、web、jpa
## 二、新建Git项目
    git init .
    git add .
    git commit -m 'first commit'
    git remote add origin https://github.com/wuyanshen/springbootjwt.git
    git push origin master
## 三、运行项目
    cd security_jwt
    mvn spring-boot:run -D maven.test.skip=true
